# YRP CREATION - Graphics Design Portfolio

## 🎨 Professional Graphics Design Portfolio Website

Welcome to the YRP CREATION portfolio website! This is a modern, responsive, and professionally designed website showcasing graphics design work, thumbnail designs, and client projects.

## ✨ Features

✅ **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
✅ **Modern UI/UX** - Clean and professional design with smooth animations
✅ **Black & Yellow Branding** - Professional color scheme matching your brand
✅ **Portfolio Gallery** - Showcase your graphics design work with filtering options
✅ **Client Showcase** - Display previously worked clients
✅ **Smooth Navigation** - Easy-to-use navigation with smooth scrolling
✅ **Contact Section** - Contact form for potential clients
✅ **Mobile-First** - Optimized for all screen sizes
✅ **Fast Loading** - Lightweight and optimized for performance

## 🎯 Sections

1. **Hero Section** - Eye-catching introduction with call-to-action button
2. **Portfolio Gallery** - Display your graphics design projects with categories:
   - All Projects
   - Thumbnails
   - Branding
   - Social Media
3. **Previously Worked Clients** - Showcase your client list
4. **About Section** - Introduction to YRP CREATION with statistics
5. **Contact Section** - Get in touch form and contact information
6. **Footer** - Social links and quick navigation

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server or database required

### Installation

1. Clone this repository:
```bash
git clone https://github.com/YubrajPaudel/yrp-creation-portfolio.git
```

2. Navigate to the project directory:
```bash
cd yrp-creation-portfolio
```

3. Open `index.html` in your web browser:
```bash
open index.html
```

Or simply double-click the `index.html` file.

## 🛠️ Customization

### Update Your Information

1. **Contact Information** (lines in `index.html`):
   - Email
   - Phone Number
   - Location

2. **Portfolio Items**:
   - Update portfolio image URLs
   - Change project titles and descriptions
   - Modify categories as needed

3. **Client Names**:
   - Edit client names and descriptions in the Clients section

4. **About Section**:
   - Update company statistics
   - Modify about text

5. **Social Links**:
   - Add your social media URLs in the footer

### Adding New Portfolio Items

```html
<div class="portfolio-item" data-category="thumbnail">
    <div class="portfolio-image">
        <img src="your-image-url" alt="Project Name">
        <div class="portfolio-overlay">
            <h3>Project Title</h3>
            <p>Project Description</p>
        </div>
    </div>
</div>
```

Categories available: `thumbnail`, `branding`, `social`

## 🌐 Deployment Options

### GitHub Pages (Free)

3. Select `main` branch as source
4. Your site will be live at: `https://yourusername.github.io/yrp-creation-portfolio`

### Netlify (Free)
1. Connect your GitHub repository
2. Deploy automatically
3. Get a free domain

### Vercel (Free)
1. Import your GitHub repository

3. Get automatic updates on push
